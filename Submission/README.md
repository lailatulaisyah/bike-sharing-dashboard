# Bike Sharing Dashboard

Dashboard interaktif berbasis Streamlit untuk menganalisis data peminjaman sepeda dari dataset Bike Sharing. Proyek ini dibuat untuk memvisualisasikan tren harian, pengaruh musim dan cuaca, serta memberikan insight operasional berdasarkan data historis.

## Setup Environment - Shell/Terminal

mkdir Submission
cd Submission
pipenv install
pipenv shell
pip install -r requirements.txt

## Menjalankan Dashboard
streamlit run dashboard.py
